# Calculator

## Classes

### Term Class
```python
  c * (x ** n)
```
Where c is a constant, \
x is the variable, \
and n is the exponent.

### Polynomial Class
```python
  (a * x ** (n)) + (b * x ** (n - 1)) + ... + (y * x ** (1)) + (z * x ** (0))
```
or
```python
  term + term + ... + term + term
```
